WFC generator

In game development WFC (Wave Function Collapse) is used for generating random output based on a set of tiles and rules. This output can then be used for different purposes, like level or texture generation. Here I'm creating textures. This is a simple Java implementation of Wave Function Collapse algorithm where the rules are defined in code.

Why: Complex result from simple rules.
Github: https://github.com/vesas/simple-wfc
Tech: Java, LibGDX, Gradle
